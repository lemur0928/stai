# STAI Library ver-0.1.0

STAI `Spatial Temporal Artificial Intelligence` 函式庫定義

## Library

- TT (Track and Trace)

    - space: 空間函式

        - cctv: CCTV影像和空間轉換算式

- CBE (Complex Behavior Expression)

    - cbe.go: 相當於 frame
    - cbeobjstruct.go: 相當於 frame 裡面的 objects
    - cbe.type.go: 裡面有 BoundingBox 和 Geo
